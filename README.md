# 🎮 Угадай число

Консольная игра на Go: нужно угадать случайное число за ограниченное количество попыток. После каждой попытки игра подсказывает, больше или меньше загаданное число, и насколько вы к нему близки.

## ⭐ Возможности

- 🎯 Три уровня сложности (Easy / Medium / Hard) с разными диапазонами и числом попыток
- 🌡️ Подсказки по расстоянию до числа: 🔥 горячо / 🙂 тепло / ❄️ холодно
- 📝 История введённых чисел
- ✅ Проверка ввода — некорректный ввод не сжигает попытку
- 🔁 Возможность сыграть ещё раз без перезапуска программы
- 💾 Сохранение результатов каждой партии в `results.json`
- 🎨 Цветной вывод в терминале (победа — зелёным, проигрыш — красным, попытки — жёлтым)

## 🎚️ Уровни сложности

| Уровень | Диапазон | Попыток |
|---------|----------|---------|
| Easy    | 1–50     | 15      |
| Medium  | 1–100    | 10      |
| Hard    | 1–200    | 5       |

## 🛠 Технологии

- **Язык:** Go (требуется версия 1.20 или новее)
- **Стандартная библиотека:** `fmt`, `math/rand`, `strconv`, `os`, `encoding/json`, `time`
- **Внешний пакет:** [`github.com/fatih/color`](https://github.com/fatih/color) — цветной вывод в терминал

## 🚀 Запуск

```bash
# Клонировать репозиторий
git clone https://github.com/AlexanderVodyanitsky/guess_the_number.git
cd guess_the_number

# Подтянуть зависимости
go mod tidy

# Запустить
go run main.go
```

Либо собрать исполняемый файл и запустить его:

```bash
go build
./guess_the_number      # Linux / macOS
guess_the_number.exe    # Windows
```

## 💾 Результаты

После каждой партии результат дозаписывается в файл `results.json` (создаётся автоматически при первом запуске):

```json
[
  {
    "date": "2026-07-01 14:30:12",
    "outcome": "победа",
    "attempts": 3,
    "level": "Medium"
  }
]
```
